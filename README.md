# 🚀 Playwright Automation Framework

## 📌 Project Overview

This project demonstrates a Playwright Automation Framework built using JavaScript. It follows the Page Object Model (POM) design pattern and includes data-driven testing, assertions, hooks, reporting, and GitHub Actions integration.

---

## 🛠️ Tech Stack

* Playwright
* JavaScript
* Node.js
* Page Object Model (POM)
* Git
* GitHub
* Allure Reports

---

## 📂 Framework Features

* Page Object Model (POM)
* Cross Browser Testing
* Data-Driven Testing using JSON & CSV
* Assertions
* Hooks (Before/After)
* Environment Variables
* Visual Testing
* Mouse & Keyboard Actions
* Drag and Drop
* GitHub Actions Integration
* Allure Reporting
